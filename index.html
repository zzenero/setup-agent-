STEP 02

Setup Termius

// SSH client untuk remote ke VPS dari HP atau PC
1
Download & install Termius
Pergi ke termius.com → download versi sesuai devicemu. Tersedia untuk Windows, macOS, Linux, iOS, dan Android. Install seperti biasa.
2
Tambah host baru
Buka Termius → klik tombol "+" atau "New Host" di halaman utama. Isi form dengan data VPS-mu:
CONFIG
Label : nama bebas (contoh: "VPS Ku") Address : 123.456.789.0 # IP address VPS Port : 22 Username : root Password : •••••••• # password dari provider
3
Connect ke VPS
Klik dua kali pada host yang sudah dibuat. Jika ada popup "host fingerprint / SSH key verification" → klik Accept atau Yes. Kamu akan masuk ke terminal Linux VPS-mu.
ℹ Kalau konek gagal, pastikan port 22 terbuka di firewall provider (biasanya ada di menu Networking atau Firewall di dashboard).
4
Update sistem terlebih dahulu
Setelah masuk terminal, wajib jalankan update dulu sebelum install apapun:
BASH
apt update && apt upgrade -y
⚠ Tunggu sampai proses selesai sebelum lanjut ke langkah berikutnya. Bisa memakan waktu 2–5 menit.
5
Install paket dasar yang diperlukan
Install git dan curl yang dibutuhkan oleh OpenClaw dan Hermes:
BASH
apt install -y git curl wget
STEP 03

Install OpenClaw

// AI assistant open source via Telegram / WhatsApp / Discord
1
Install Node.js v20
OpenClaw butuh Node.js. Jalankan dua perintah ini satu per satu:
BASH
curl -fsSL https://deb.nodesource.com/setup_20.x | bash - apt install -y nodejs
ℹ Verifikasi: node --version — harus menampilkan v20 atau lebih tinggi.
2
Install OpenClaw secara global
Jalankan perintah install ini:
BASH
npm i -g openclaw
3
Siapkan API key dan Bot Token dulu
Sebelum onboarding, siapkan dua hal ini:
CHECKLIST
✓ API Key dari Anthropic (claude.ai/settings/keys) atau OpenRouter (openrouter.ai/keys) atau OpenAI (platform.openai.com/api-keys) ✓ Bot Token Telegram (jika pakai Telegram): → Buka Telegram → cari @BotFather → Ketik /newbot → ikuti instruksi → Copy token yang diberikan
4
Jalankan wizard onboarding
Jalankan perintah setup pertama kali. Wizard akan memandu kamu secara interaktif:
BASH
openclaw onboard
YANG AKAN DITANYAKAN
Nama AI : bebas (contoh: Jarvis, Claw, dll) API Key : paste API key yang sudah disiapkan Platform : pilih Telegram / WhatsApp / Discord Bot Token : paste token dari @BotFather
5
Jalankan OpenClaw
Setelah onboarding selesai, start OpenClaw:
BASH
openclaw start
ℹ Agar tetap berjalan walau Termius ditutup: nohup openclaw start & — atau gunakan screen / tmux.
6
Test di Telegram / WhatsApp
Buka platform chat yang kamu pilih, cari bot yang sudah dibuat, kirim pesan "halo". Bot AI-mu harus sudah merespons! Selamat, OpenClaw aktif 🎉
STEP 04

Install Hermes Agent

// by Nous Research — lebih powerful, self-learning, bisa migrate dari OpenClaw
1
Install Hermes dengan satu perintah
Installer otomatis ini akan mengurus Python, Node.js, dan semua dependensi sendiri. Tidak perlu install apapun dulu:
BASH
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
⚠ Butuh git terinstall. Jika belum: apt install -y git
2
Reload shell setelah install
Setelah installer selesai, reload terminal agar perintah hermes dikenali:
BASH
source ~/.bashrc # atau jika pakai zsh: source ~/.zshrc
3
Jalankan setup wizard lengkap
Setup wizard interaktif yang akan mengkonfigurasi API key, model, dan platform sekaligus:
BASH
hermes setup
YANG AKAN DIKONFIGURASI
Provider model : OpenRouter / Anthropic / OpenAI / Nous Portal API Key : paste API key provider yang dipilih Platform : Telegram / Discord / Slack / WhatsApp / Signal Bot Token : token bot platform yang dipilih
ℹ Jika wizard mendeteksi folder ~/.openclaw, ia akan menawarkan migrasi otomatis dari OpenClaw. Jawab yes!
4
Setup dan mulai gateway messaging
Agar Hermes bisa diakses dari Telegram/Discord/WhatsApp:
BASH
hermes gateway setup # konfigurasi platform hermes gateway start # mulai gateway
5
Perintah-perintah berguna Hermes
Perintah dasar yang sering dipakai:
BASH
hermes # mulai chat via terminal (CLI) hermes model # ganti model AI hermes tools # konfigurasi tools yang aktif hermes doctor # diagnosa masalah instalasi hermes update # update ke versi terbaru

STEP 05

Migrate OpenClaw → Hermes

// pindah semua settings, memory, skills, dan API keys otomatis
// APA YANG DIPINDAHKAN
  • SOUL.md (persona / karakter AI)
  • Memories (MEMORY.md & USER.md)
  • Skills yang kamu buat sendiri
  • API keys (Telegram, OpenRouter, Anthropic, dll)
  • Konfigurasi platform messaging
  • Command allowlist
  • TTS assets & workspace audio
// PERINTAH MIGRATE
  • hermes setup (otomatis detect OpenClaw)
  • hermes claw migrate (manual)
  • hermes claw migrate --dry-run (preview)
  • hermes claw migrate --overwrite
  • hermes claw migrate --preset user-data
1
Cara termudah: lewat hermes setup
Saat pertama kali jalankan hermes setup, wizard akan otomatis mendeteksi folder ~/.openclaw dan menawarkan migrasi. Cukup jawab yes.
2
Preview dulu sebelum migrate (aman)
Kalau mau lihat apa yang akan dipindahkan tanpa benar-benar eksekusi:
BASH
hermes claw migrate --dry-run
3
Jalankan migrasi penuh
Setelah yakin dengan hasil preview, jalankan migrasi sesungguhnya:
BASH
hermes claw migrate
4
Update Hermes secara rutin
Hermes aktif dikembangkan dengan update hampir setiap hari. Selalu update untuk dapat fitur dan bug fix terbaru:
BASH
hermes update
ℹ Kalau ada masalah setelah update: hermes doctor — untuk diagnosa otomatis.
